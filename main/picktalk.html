1 휴대폰에 설치 (홈화면 추가)

아이콘으로 열어야 일반 앱처럼 동작하고, 알림도 제대로 옵니다.

2 알림 허용

아래 큰 버튼을 누르면 iOS/안드로이드가 알림 허용 팝업을 띄웁니다.
팝업이 뜨면 "허용" 을 선택하세요.

⚠️ 알림이 차단됨 상태입니다.
iOS: 설정 앱 → 알림 → 픽앤톡 → "알림 허용" 켜기
Android: 브라우저 설정 → 사이트 권한 → 알림 → makewon.com 허용
✓ 알림이 켜져 있습니다. 위 버튼을 다시 눌러 테스트 알림을 받을 수 있습니다.

3 어떻게 동작하나요?

설치 + 알림 허용이 끝나면, 동료가 작업을 등록할 때 휴대폰 잠금화면에 알림이 옵니다. 알림을 누르면 픽앤톡 앱이 열리며 새 작업이 바로 보입니다.

※ 실제 푸시 발송 기능은 단계적으로 활성화됩니다. 지금은 설치·알림 권한 까지 준비해두시면 됩니다.

❓ 자주 묻는 질문

Q. iOS 에서 "홈 화면에 추가" 가 어디 있나요?
Safari 하단 가운데 공유 버튼(⬆️ 화살표) → 메뉴 스크롤 → "홈 화면에 추가".

Q. 알림이 안 와요
① 홈 화면의 픽앤톡 아이콘으로 열었는지 (일반 Safari 아님)
② 휴대폰 설정 → 픽앤톡 → 알림 켜져 있는지
③ 방해 금지 모드(집중 모드) 해제했는지

Q. 카카오톡 알림도 받을 수 있나요?
별도 카카오 알림톡 연동이 필요합니다 (사업자 채널 + 중개사 가입). 우선 휴대폰 자체 알림으로 시작하시고, 필요하면 카카오 연동을 추가합니다.
