라벨 추출

사진 일괄 업로드 → 바코드·Style Code·사이즈 추출 → 파일명 일괄 정리 → manifest.csv + ZIP 다운로드

📌 작업 순서   ① 📁 폴더 선택 → 사진 전부 자동 등록 + ZXing 바코드 자동 스캔 ② (선택) 📋 Claude로 분석하기 → 텍스트 필드 자동 채움 ③ 카드 이미지 클릭 → 원본 확대 + 7필드(바코드/품명/품번/사이즈/색상/재질/제조국) 수동 입력/수정 ④ ✓ 완료 확정 → 다음 사진으로 자동 이동 (← / → 키로도 이동 가능) ⑤ 💾 manifest.csv + 🗜️ ZIP 다운로드
🔍 바코드 정책  ZXing(수학 디코딩) 결과 = 권위. Claude OCR 결과는 cross-check 용. 불일치 시 barcode_mismatch 경고.
🏷️ 파일명 규칙  {바코드}_{StyleCode}_{사이즈}.{확장자}  예: 887224608515_881224-100_235.jpg
🎯 신뢰도  ≥0.85 OK · 0.6~0.85 검수(노랑) · <0.6 실패
💳 비용  ZXing은 무료(로컬 처리) · Claude 텍스트 추출은 claude.ai 구독 안에서
⚠️ 중복 바코드는 _01, _02 자동 부여. HEIC 미지원.
