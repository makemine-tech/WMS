📌 작업 순서   ① 📁 폴더 선택 → 사진들 들어있는 폴더 1번 선택하면 전부 자동 등록 (개별 드래그 불필요) ② 📋 Claude로 분석하기 → 모달에서 프롬프트 복사 → claude.ai 채팅에 사진 함께 첨부 → JSON 응답 받기 ③ 응답 JSON을 모달의 텍스트영역에 붙여넣고 적용 ④ 카드 확인/수정 (검수 우선 = 노란 테두리) ⑤ 💾 manifest.csv + 🗜️ ZIP 다운로드

⚠️ 중복 바코드는 _01, _02 시퀀스 자동 부여. HEIC 미지원(JPG/PNG/WEBP).
